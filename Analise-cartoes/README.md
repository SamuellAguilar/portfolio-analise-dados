# 📊 Análise de Cancelamento de Clientes (Churn)

## 📌 Sobre o projeto

Este projeto tem como objetivo analisar dados de clientes de um banco para identificar os principais fatores que levam ao cancelamento de cartões de crédito (**churn**).

A análise busca entender padrões de comportamento dos clientes e gerar insights que possam auxiliar na **retenção e tomada de decisão estratégica**.

---

## 🎯 Objetivo

Identificar variáveis e comportamentos associados ao cancelamento de clientes, respondendo perguntas como:

* Quais fatores influenciam o churn?
* Clientes com mais produtos cancelam menos?
* O uso do cartão impacta a retenção?
* Existem perfis de clientes com maior risco de cancelamento?

---

## 🛠️ Ferramentas utilizadas

* Python
* Pandas
* Plotly
* Jupyter Notebook / Google Colab

---

## 📊 Análises realizadas

Durante o projeto foram realizadas:

* Análise exploratória de dados (EDA)
* Visualização de distribuições das variáveis
* Comparação entre clientes ativos e cancelados
* Identificação de padrões de comportamento

---

## 🔍 Principais insights

### 📌 1. Produtos e retenção

Clientes com maior número de produtos apresentam menor taxa de cancelamento.

👉 Indica que clientes mais engajados tendem a permanecer.

---

### 📌 2. Uso do cartão

Clientes com maior volume de transações tendem a cancelar menos.

👉 O uso frequente está diretamente relacionado à retenção.

---

### 📌 3. Perfil de risco

Clientes com baixo nível de atividade apresentam maior probabilidade de churn.

👉 Esses clientes devem ser monitorados de forma preventiva.

---

## 💡 Aplicações de negócio

A partir dos insights obtidos, é possível:

* Criar campanhas de retenção para clientes inativos
* Incentivar o uso do cartão
* Oferecer novos produtos para aumentar engajamento
* Identificar clientes com risco de cancelamento

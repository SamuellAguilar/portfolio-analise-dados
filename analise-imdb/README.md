# PROJETO FINAL – ANÁLISE DE DADOS IMDB TOP 1000

## 1. DESCRIÇÃO DO PROJETO

Este projeto foi desenvolvido como atividade final do curso de Analista de Dados. O objetivo foi aplicar na prática as etapas de um processo completo de análise de dados, incluindo seleção da base de dados, análise exploratória, tratamento dos dados e construção de visualizações.

Foi utilizado o dataset público **"IMDB Top 1000 Movies"**, que contém informações sobre os mil filmes mais bem avaliados no site IMDb.

---

## 2. BASE DE DADOS

O conjunto de dados utilizado contém informações como:

- Título do filme  
- Ano de lançamento  
- Gênero  
- Duração do filme  
- Avaliação no IMDb  
- Número de votos  
- Metascore  
- Receita bruta (Gross)

A base possui mais de **1000 registros** e diversas colunas com informações relevantes para análise.

---

## 3. ETAPAS DO PROJETO

### Seleção dos Dados

Foi selecionado o dataset **IMDB Top 1000**, disponível publicamente em plataformas de dados abertos.

### Análise Exploratória

Foram realizadas análises para compreender o comportamento dos dados, incluindo:

- Distribuição de filmes por ano de lançamento  
- Identificação dos gêneros mais frequentes  
- Relação entre avaliação do IMDb e número de votos  
- Comparação entre duração dos filmes  
- Identificação dos filmes mais bem avaliados

### Tratamento e Preparação dos Dados

Nesta etapa foram realizados:

- Tratamento de valores inconsistentes  
- Padronização de tipos de dados  
- Conversão de colunas numéricas  
- Criação de novas colunas derivadas para auxiliar nas análises

### Visualização de Dados

Os resultados da análise foram apresentados por meio de gráficos e visualizações interativas utilizando o Looker Studio.

Foi desenvolvido um dashboard contendo diferentes gráficos que permitem identificar padrões e tendências nos dados.
